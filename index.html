Introduction

Welcome to the Programming Tutorial: HTML, CSS, and JavaScript

Objective of the tutorial:

In this tutorial, you will learn the fundamentals of HTML, CSS, and JavaScript, the three pillars of web development.

By the end, you will be able to create an interactive and styled web page.

Prerequisites
  • Basic computer knowledge.
  • A computer with internet access.
  • A text editor (recommended: Visual Studio Code).
Installation and Setup
  • Installing the development environment:
    • Download and install Visual Studio Code.
    • Install the recommended extensions: Live Server, Prettier.
  • Initial setup:
    • Open Visual Studio Code.
    • Set up the environment by following these steps:
    • Open a new folder for your project.
    • Create an index.html, styles.css, and script.js file.
Basic HTML

  • Structure of an HTML document:


                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>My Web Page</title>
                    <link rel="stylesheet" href="styles.css">
                </head>
                <body>
                    <h1>Hello, world!</h1>
                    <script src="script.js"></script>
                </body>
                </html>
  • Selectors and properties:
    • Class selectors: .class { property: value; }
    • ID selectors:#id { property: value; }
    • Element selectors:element { property: value; }
    Basic JavaScript
    • Basic JavaScript syntax:
      • console.log("Hello, world!");
    • Variables and data types:
      • let name = "John";
        let age = 30;
        let isStudent = true;
    • Functions:
      • function greet(name) {
        return "Hello, " + name;
        }

        let message = greet("John");
        console.log(message); // Output: Hello, John
    Interactivity with JavaScript
    • DOM manipulation:
      • document.getElementById("myElement").innerHTML = "New content";
    • Events:
      • document.getElementById("myButton").addEventListener("click", function() {
        alert("Button clicked!");
        });
    Practical Projects
    • Project 1: Personal Introduction Page
      • Create a page with your personal information, including a photo, a brief biography, and links to your social media.
    • Project 2: Basic Calculator
      • Develop a calculator that can perform basic operations like addition, subtraction, multiplication, and division.
    • Project 3: To-Do List
      • Create an application that allows adding, deleting, and marking tasks as completed.
